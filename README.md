# oedisi-ieee123-main

# Интерактивный симулятор распределительной сети IEEE 123

Для тех, кому лень много читать: Для симуляции рабобы узлов просто выбираете режим его работы и один раз кликаете на узле для получения результатов симуляции. )))

Общее описание: Проект представляет собой программный комплекс, предназначенный для моделирования, визуализации и анализа режимов работы стандартной распределительной электрической сети IEEE 123 Bus. В качестве расчетного ядра используется движок OpenDSS (через библиотеку dss-python).

Инструмент позволяет наглядно исследовать поведение распределительной сети при различных эксплуатационных и аварийных сценариях, а также оценивать влияние распределенной генерации на параметры режима.

# Ключевые функциональные возможности

Визуализация топологии: Построение интерактивного графа сети с цветовой кодировкой узлов (нагрузки, регуляторы, солнечные панели). Реализована навигация и управление параметрами отображения.

Сценарное моделирование аварий: Возможность создания пользователем аварийных возмущений (короткие замыкания, обрывы линий) в конкретных узлах и фазах в режиме реального времени.

Анализ QSTS (Quasi-Static Time Series): Расчет суточных профилей напряжения и тока с шагом 15 минут. Используются модифицированные модели нагрузок (ZIP-модель) для учета зависимости потребления от напряжения.

Интеграция возобновляемых источников (ВИЭ): Моделирование работы солнечных панелей (PV) с учетом суточных профилей инсоляции и температурных коэффициентов.

Мониторинг качества работы сети: Автоматический анализ результатов симуляции для выявления узлов с нарушениями уровней напряжения (просадки ниже 0.95 p.u. или перенапряжения выше 1.05 p.u.).

Техническая реализация

Стек технологий: Python 3.10 (или 3.11), OpenDSS (расчет режимов), Matplotlib (графический интерфейс и построение графиков), Pandas/Numpy (обработка данных).

Архитектура: Модульная структура, включающая скрипт запуска (main.py), модуль графического интерфейса (plot_topology.py) и расчетный модуль (run_qsts_plot.py). Физическая модель сети описана в DSS-файлах с подключением внешних CSV-профилей нагрузок.
